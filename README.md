# C-Gate Server — Home Assistant Add-on

Home Assistant add-on for the Schneider Electric SpaceLogic C-Gate Server,
providing C-Bus home automation network management.

## Features

- C-Gate Server v3.7.0 (build 2285)
- Built-in web console with real-time C-Bus event streaming
- Home Assistant ingress integration
- Configurable project, interface IP, and log level
- Multi-architecture: amd64 and aarch64

## Installation

Add this repository to Home Assistant:

1. Go to **Settings → Add-ons → Add-on Store**
2. Click **⋮ → Repositories**
3. Paste: `https://github.com/rbhr/ha-app-C-Gate-Server`
4. Find **C-Gate Server** and click **Install**
